발표 영상

개발 동기

밀리미터파는 도심의 복잡한 환경에 쉽게 영향을 받아 기존의 실증적인 방법을 이용하여 경로 손실을 예측하는 데에 많은 계산량 등의 한계가 존재한다

개발 목표

도심지에서 CNN을 기반으로 28 GHz 대역폭의 밀리미터 경로 손실 모델 방법을 제시한다.
3D-LAMS를 이용하여 CNN의 입력 이미지를 만들어 경로 손실 환경 정보를 모델의 학습에 이용한다.

향후 전망

취약 범위도 통신을 원활하도록 기지국을 설치하여 사용자 경험 개선
